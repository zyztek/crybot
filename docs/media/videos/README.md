## Video Production Standards

### Technical Specifications
- **Resolution**: 1920x1080 (Full HD) minimum
- **Frame Rate**: 30fps (60fps for high-motion content)
- **Bitrate**: 8-12 Mbps (1080p), 4-6 Mbps (720p)
- **Codec**: H.264 (MP4), VP9 (WebM)
- **Audio**: AAC, 44.1kHz, 128kbps minimum
- **Format**: MP4 (primary), WebM (web optimization)

### Audio Requirements
- **Voice Quality**: Clear, professional narration
- **Background Music**: Royalty-free, subtle
- **Sound Effects**: Professional, non-distracting
- **Volume Levels**: Balanced voice/music/effects
- **Accessibility**: Subtitles and transcripts available

### Visual Standards
- **Branding**: Consistent Crybot branding
- **Typography**: Inter font family
- **Colors**: Brand color palette
- **Graphics**: High-quality, professional
- **Animations**: Smooth, purposeful
