# Crybot Logo Suite

## Logo Variations

### Primary Logo
- **Format**: SVG, PNG (transparent background)
- **Colors**: Full color version
- **Usage**: Main branding, headers, marketing materials
- **Minimum Size**: 32px height
- **Clear Space**: 2x logo height

### Secondary Logo
- **Format**: SVG, PNG (transparent background)
- **Colors**: Monochrome version
- **Usage**: Small spaces, watermarks, secondary branding
- **Minimum Size**: 24px height
- **Clear Space**: 1.5x logo height

### Icon Logo
- **Format**: SVG, PNG (transparent background)
- **Colors**: Single color version
- **Usage**: Favicon, app icons, small elements
- **Sizes**: 16px, 24px, 32px, 48px, 64px, 128px, 256px
- **Applications**: Web, mobile, desktop

## Logo Components

### Symbol
- **Design**: Abstract cryptocurrency faucet with blockchain elements
- **Meaning**: Represents the flow of cryptocurrency and blockchain connectivity
- **Elements**: Water droplet (faucet), hexagon (blockchain), arrows (flow)

### Typography
- **Font**: Inter, Bold
- **Style**: Modern, clean, tech-forward
- **Letter Spacing**: Slightly condensed for impact
- **Weight**: Bold for primary, Medium for secondary

## Usage Guidelines

### Do's
- Use appropriate logo variation for context
- Maintain clear space around logo
- Ensure sufficient contrast with background
- Scale proportionally
- Use high-resolution versions

### Don'ts
- Stretch or distort logo
- Change logo colors (except approved variations)
- Add effects or filters
- Place logo on busy backgrounds
- Use logo smaller than minimum size

## Technical Specifications

### SVG Format
- **ViewBox**: 0 0 100 30 (for horizontal logo)
- **Paths**: Optimized for web
- **Colors**: Hex values for consistency
- **Text**: Converted to paths for compatibility

### PNG Format
- **Resolution**: 300 DPI for print, 72 DPI for web
- **Compression**: Optimized for file size
- **Transparency**: Alpha channel support
- **Color Profile**: sRGB for web

### Icon Formats
- **Favicon**: ICO format with multiple sizes
- **App Icons**: PNG format with rounded corners
- **Social Icons**: PNG format with platform-specific sizes
