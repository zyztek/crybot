## Presentation Templates

### Master Template
- **Design**: Consistent with Crybot brand guidelines
- **Colors**: Primary (#6366f1), Secondary (#8b5cf6), Accent (#ec4899)
- **Typography**: Inter font family
- **Layout**: 16:9 aspect ratio (1920x1080)
- **Elements**: Logo, title, content, footer

### Slide Structure
```
1. Title Slide
   - Logo
   - Presentation title
   - Subtitle
   - Presenter name
   - Date

2. Agenda Slide
   - Overview of topics
   - Timeline
   - Key takeaways

3. Content Slides
   - Clear headings
   - Concise bullet points
   - Visual elements
   - Data visualization

4. Summary Slide
   - Key points recap
   - Call to action
   - Contact information

5. Q&A Slide
   - Discussion prompts
   - Contact details
   - Thank you message
```

## Design Guidelines

### Typography
- **Title**: Inter Bold, 48-60pt
- **Subtitle**: Inter Medium, 32-36pt
- **Body**: Inter Regular, 24-28pt
- **Caption**: Inter Light, 18-20pt

### Colors
- **Primary Text**: #1f2937 (Dark Gray)
- **Secondary Text**: #6b7280 (Medium Gray)
- **Accent Color**: #6366f1 (Purple)
- **Background**: #ffffff (White)
- **Highlight**: #ec4899 (Pink)

### Visual Elements
- **Charts**: Consistent color scheme
- **Icons**: Lucide React style
- **Images**: High-quality, relevant
- **Diagrams**: Clean, professional

## Content Guidelines

### Writing Style
- **Tone**: Professional, confident, engaging
- **Language**: Clear, concise, jargon-free
- **Voice**: Active, direct, authoritative
- **Length**: 6-8 words per bullet point

### Data Presentation
- **Charts**: Bar charts for comparisons, line charts for trends
- **Numbers**: Use commas for thousands, 2 decimal places
- **Percentages**: Include context and comparison
- **Sources**: Cite data sources in footers

### Storytelling
- **Structure**: Problem, solution, benefit
- **Flow**: Logical progression
- **Emotion**: Connect with audience
- **Call to Action**: Clear next steps
