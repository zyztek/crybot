## Production Guidelines

### Brand Guidelines
- **Primary Colors**: #6366f1 (Purple), #8b5cf6 (Violet), #ec4899 (Pink)
- **Typography**: Inter, sans-serif
- **Logo Usage**: Minimum 32px height, clear space 2x logo height
- **Image Quality**: Minimum 72 DPI for web, 300 DPI for print

### Video Specifications
- **Resolution**: 1920x1080 (Full HD)
- **Format**: MP4, WebM
- **Frame Rate**: 30fps
- **Duration**: 30 seconds to 5 minutes
- **Audio**: Stereo, 44.1kHz, 128kbps minimum

### Presentation Templates
- **Aspect Ratio**: 16:9 (1920x1080)
- **Font Size**: Minimum 24pt for body text
- **Color Contrast**: WCAG 2.1 AA compliance
- **File Format**: PDF, PowerPoint, Google Slides
